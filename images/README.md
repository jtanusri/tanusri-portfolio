## Image Guidelines

- **Format**: PNG with transparent background preferred, JPG also works
- **Size**: 120x120px recommended (will be displayed at ~48-56px)
- **Aspect Ratio**: Square (1:1) works best
- **File Size**: Keep under 50KB each for fast loading

## Dark Mode Support

You have two options for dark mode:

### Option 1: Single Logo (works on both backgrounds)
Just provide the standard logo file (e.g., `nfl.png`). It will be used for both light and dark modes.

### Option 2: Separate Light/Dark Logos
Provide two versions:
- `nfl.png` - Used in light mode (light backgrounds)
- `nfl-dark.png` - Used in dark mode (dark backgrounds)

The website will automatically switch between them based on the user's theme.

### Tips for Dark Mode Logos
- Use lighter colors or white versions for dark mode
- Ensure sufficient contrast against dark backgrounds (#0f172a)
- Some logos with dark text may need an inverted or white version
